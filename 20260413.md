## 今日遇到的报错/异常

1. 终端会话中断
- 现象：执行串联验证命令时出现终端被关闭、命令退出 `130`。
- 原因：多段命令中含中断/超时流程，导致会话被提前终止。
- 处理：改为分步执行；出现残留进程时用 `pkill` 清理。

2. 话题采样时短暂告警
- 现象：`ros2 topic hz /mavros/local_position/pose` 提示 `topic does not appear to be published yet`。
- 原因：采样开始早于系统稳定拉起时刻。
- 处理：增加启动等待时间并通过 launch 日志确认节点已运行。

3. `KeyboardInterrupt`（Perception 节点）
- 现象：手动 `Ctrl-C` 结束联调时，Python 节点抛出中断栈。
- 结论：正常退出路径，不是功能性缺陷。

4. `get_changed_files` 提示非 git 仓库
- 现象：工具返回当前目录不是 git 仓库。
- 处理：改用直接文件核对与构建验证，不影响本次实现。
